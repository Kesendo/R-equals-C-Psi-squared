## Dephasing axis dependence (N=3 chain, γ=0.05, the Z-dephasing Π)

The residual is the max entry of Π·L·Π⁻¹ + L + 2Σγ·I IN THE PAULI-STRING
BASIS (a max entry is basis-dependent, so the basis is part of the number).
'complex' pairs λ ↦ −λ−2Σγ, the theorem's statement; 'rates' pairs d ↦ 2Σγ−d.

                      Axis   Π·L_H   Π·L_D   residual    complex      rates
----------------------------------------------------------------------------
                         Z       ✓       ✓     0.0000      64/64      64/64
                         Y       ✓       ✓     0.0000      64/64      64/64
                         X       ✓       ✗     0.3000      64/64      64/64
                  mixed ZX       ✓       ✗     0.1000      64/64      64/64
 depolarizing (γ per axis)       ✓       ✗     0.9000       0/64       0/64
depolarizing (γ/3 per axis)       ✓       ✗     0.1000       0/64      14/64

Σγ = 0.15. The two depolarizing rows differ only in convention:
  γ per axis   → residual = 6·Σγ = 0.90
  γ/3 per axis → residual = (2/3)·Σγ = 0.1000  (the registered F1 value)

The residual is a MAX ENTRY, which is basis-dependent, not a norm.
Same operator, γ/3 depolarizing row, two bases:
  Pauli-string basis: 0.1000 = (2/3)·Σγ = 0.1000
  computational basis: 0.0333 = (2/9)·Σγ = 0.0333

The residual is |−8Nγs + 2Nγ| for scale s, so the ratio 9 between the two
rows is NOT 'three times the rate': the +2Σγ offset is not tripled with it.
Both break the palindrome outright: the complex pairing is 0/64 in BOTH
conventions. The γ/3 row's surviving rate-only count is the weaker reading,
which tolerates breakage the theorem's statement does not.

## Scope boundaries (N=3 chain unless noted, γ=0.05)

Each row is a claim the proof document makes about what lies OUTSIDE
the theorem. 'complex' pairs λ ↦ −λ−2Σγ; 'rates' pairs d ↦ 2Σγ−d.

                              Case   residual    complex      rates
--------------------------------------------------------------------
            bond-only H (in scope)     0.0000      64/64      64/64
   + longitudinal Z field, uniform h=0.3     0.6000       0/64      64/64
     + longitudinal Z field, non-uniform     1.0000       0/64      28/64
       + transverse X field, non-uniform     0.0000      64/64      64/64
       + transverse Y field, non-uniform     1.0000      64/64      64/64

The Y row's nonzero residual is a GAUGE artifact, not a property of the
Y direction. Rotating the field in the XY plane by an angle φ away from X,
the residual under this FIXED Π traces 2·max|h|·|sin φ|:

  φ (deg)   residual   2·max|h|·|sin φ|
----------------------------------------
        0   0.000000           0.000000
       15   0.258819           0.258819
       30   0.500000           0.500000
       45   0.707107           0.707107
       60   0.866025           0.866025
       90   1.000000           1.000000

A per-site R_z(π/2) maps the X-field Hamiltonian to the Y-field one and
is a symmetry of Z-dephasing, so the two Liouvillians are unitarily
equivalent and their spectra are identical. Conjugating Π by the same
rotation satisfies the identity exactly for the Y field. So the residual
under a fixed Π grades the ANGLE to that mirror's preferred axis.

But that argument needs ONE common direction: a global rotation cannot
align several at once. Let the direction vary per site and it breaks.

         per-site field directions   residual    pairing
----------------------------------------------------------
                       all along X     0.0000      64/64
                       all along Y     1.0000      64/64
                     all at 40 deg     0.6442      64/64
                           X, Y, X     0.4000       4/64
                    0, 40, 109 deg     0.9463       0/64

Uniform direction, any angle: exact. Direction varying between sites:
gone. Magnitudes may vary freely in both cases. The general residual law
is 2·maxᵢ|hᵢ·sin φᵢ|, which collapses to 2·max|h|·|sin φ| for a common
angle. This rhymes with the dephasing-axis rule below, where mixing axes
across sites is also the thing that eventually breaks the palindrome.

ONE rule covers both field directions: the field survives exactly when
its axis is ORTHOGONAL to every dephasing axis in play. Avoiding the
used axes is not enough, as the 45° rows below show. 'Transverse' and
'longitudinal' are not two facts, they are this one fact seen from a
Z-dephasing default. Every axis-set and field-direction combination:

  dephasing axes present   field X   field Y   field Z
------------------------------------------------------
                       Z        64        64         0
                     X+Z         0        64         0
                     Y+Z        64         0         0
                     X+Y         0         0        64
                     X+Y         0         0        64
                     Y+Z        64         0         0

Every 64 sits orthogonal to the noise's axes, every 0 does not. Note
that AVOIDING the used axes is not sufficient, only orthogonality is:

     dephasing        field direction    pairing
--------------------------------------------------
             Z                along X      64/64
             Z    45° in the XY plane      64/64
           X+Z                along Y      64/64
           X+Z    45° in the XY plane       0/64

The 45° field lies along neither Z nor X, yet dies once both are
dephasing: it is not orthogonal to X. With Z alone the whole plane is
orthogonal to the noise, so the same 45° field is exact.

### The boundary is sharp, the failure is not

Chain dephased along Z with an X field (legal), then a competing
X-dephasing switched on at rate γ_X on one site (forbidden). The
pairing degrades with γ_X rather than collapsing at the first
nonzero value, so a small violating term costs proportionally little:

       γ_X   pairs @1e-7   pairs @1e-10
----------------------------------------
     0e+00         64/64          64/64
     1e-12         64/64          64/64
     1e-09         64/64          20/64
     1e-06         20/64           0/64
     1e-03          0/64           0/64
     5e-02          0/64           0/64

A measurement can only see a violation larger than its own tolerance.
The rule says where the palindrome is EXACT, not that everything
outside is equally far away.

And the surviving rate pairing under the longitudinal field comes from
UNIFORMITY, not from U(1) conservation: a uniform h·ΣZ commutes with H
and with the dephasing, so it enters as a pure imaginary shift. A
non-uniform longitudinal field is just as U(1)-conserving and wrecks the
rate pairing too, as the second row shows.

### Which label permutations admit a working Π at all

All 24 permutations of {I,X,Y,Z} × all 256 phase assignments in {±1,±i}:

  I→X X→I Y→Z Z→Y    16/256 phases work
  I→X X→Z Y→I Z→Y    16/256 phases work
  I→Y X→I Y→Z Z→X    16/256 phases work
  I→Y X→Z Y→I Z→X    16/256 phases work

Exactly these four, and they are exactly the four swapping the sets
{I,Z} and {X,Y}: two involutions and two 4-cycles. That swap IS the
XY-weight flip w → N−w of Step 1, so the condition is structural, not
a lucky choice of labels. No purely real phase assignment works.
